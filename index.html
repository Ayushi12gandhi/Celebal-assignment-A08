🌿 Git Commands Dashboard

Your Git cheat sheet for everyday development!

📦 Stage & Commit

git add . git commit -m "your message"

🔀 Switch Branch

git checkout -b new-feature

🚀 Push to Remote

git push origin new-feature

🧯 Restore Deleted File

git checkout HEAD^ -- index.html

⏪ Reset to Commit

git reset --hard <commit-hash>

🤝 Merge Conflict Fix

git merge main // Resolve conflicts manually git add . git commit
